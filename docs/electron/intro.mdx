## Electron是什么？
Node很强大 拥有操作io、访问系统底层API等高级权限。       
Web用于布局且很快，远比 Windows原生C语言和 MacOs原生Swift布局效率高。     

两者强强联合，我们就能快速开发桌面端应用，而Electron就是干这个事情的：

> Electron是一个使用 JavaScript、HTML 和 CSS 构建桌面应用程序的框架。 嵌入 Chromium 和 Node.js 到 二进制的 Electron 允许您保持一个 JavaScript 代码代码库并创建 在Windows上运行的跨平台应用 macOS和Linux——不需要本地开发 经验。


**用一句时髦的话来说：elctron给web赋能，让端开发者不再仅仅局限于能写网站网页。**
